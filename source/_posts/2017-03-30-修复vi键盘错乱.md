**原因：ubuntu vi 默认是安装 vim-tiny**
#### 1、卸载
{% codeblock lang:bash %}
sudo apt-get remove vim-common
{% endcodeblock %}
#### 2、安装
{% codeblock lang:bash %}
sudo apt-get install vim
{% endcodeblock %}
